## OpenGL Sandbox

Project created to learn OpenGL (incl. GLSL, GLFW, GLAD, GLM)

### Prerequisites

* [CMake](https://cmake.org/install) v3.16+

### Installation

* Clone the repo with submodules

```bash
git clone --recurse-submodules https://github.com/pavelgrin/opengl_sandbox
```

* Build project

```bash
mkdir build && cd "$_"
cmake ..
cmake --build .
```

* Run

```bash
build/bin/app
```
